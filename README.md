
## Description

Backend for github rest api commit history app

## Installation

- create a .env file with the next field and  submitted api key
### GITHUB_ACCESS_TOKEN=  "api key"

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
